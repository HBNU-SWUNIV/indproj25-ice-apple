## Project Background
- ### 필요성
	•	기존 학과 홍보, 프로젝트 안내, 회의실 예약 등은 포스터·브로슈어 같은 정적 방식에 의존 → 실시간 정보 전달이 어려움
	•	정보 관리가 각 부서/학생에 개별적으로 분리되어 있어 유지보수/관리 비효율
	•	학과 행사, 졸업작품, 연구성과 등을 한 플랫폼에서 통합적으로 보여줄 시스템 필요
	•	다양한 기능을 유연하게 확장 가능한 구조(MSA 기반) 요구
- ### 기존 해결책의 문제점
 •	포스터, 공지판 기반 정보 전달 : 실시간 업데이트 불가, 관심 유도 어려움
 •	카카오톡/구글폼 회의실 예약 : 수동 관리, 예약 중복 발생 가능
 •	단일 서버 구조 키오스크 : 기능 추가 시 전체 시스템 수정 필요, 유지보수 어려움
 •	영상 재생형 홍보 키오스크 : 인터랙션 부족, 데이터 저장 및 분석 불가능
  
## System Design
본 시스템은 MSA(Microservice Architecture) 기반으로 설계되었으며, 공지사항, 회의실 예약, 프로젝트 전시, 홍보영상, 인증 서비스 등 기능들을 각각 독립적인 마이크로서비스로 구현하였다. 각 서비스는 개별적으로 개발 및 배포가 가능하며, API Gateway를 통해 통합적으로 관리된다. 백엔드는 Spring Boot를 기반으로 하고, 프론트엔드는 React로 개발했으며, 데이터 저장은 PostgreSQL을 사용하였다. 또한 Docker와 GitHub Actions를 활용하여 서비스별 CI/CD 자동 배포 환경을 구축하였고, Raspberry Pi 환경에서 실제 키오스크 형태로 운용할 수 있도록 설계하였다.
    
  - ### System Requirements
     본 시스템이 만족해야 할 요구사항은 다음과 같다.
각 기능은 MSA 구조로 분리되어 독립적인 개발과 유지보수가 가능해야 하며, 관리자 페이지에서 공지, 영상, 예약, 프로젝트 정보를 통합적으로 관리할 수 있어야 한다. 또한 사용자 화면은 터치 기반 UI로 구성되어 직관적인 정보 전달이 가능해야 하며, PostgreSQL을 기반으로 모든 서비스 데이터가 안정적으로 관리되어야 한다. 배포 환경은 Docker, GitHub Actions, Raspberry Pi를 활용하여 실제 키오스크 장치에서 실행 가능한 형태를 갖춰야 한다.
    
## Case Study
  - ### Description
     본 프로젝트는 실제 학과 환경에서 활용 가능한 키오스크 시스템을 목표로 하며, 학과 홍보, 회의실 예약, 영상 재생, 공지사항 전달 등의 기능을 하나의 플랫폼에서 운영할 수 있도록 설계되었다. 특히 기존에 포스터, 브로슈어, 개별 페이지 등으로 분산되어 있던 정보 전달 방식을 통합 디지털 플랫폼으로 전환하였으며, 마이크로서비스 구조를 적용해 기능별 확장이 용이하도록 구현하였다. Raspberry Pi를 활용하여 실물 키오스크 형태로 운영할 수 있도록 적용하였으며, GitHub Actions 기반 자동 배포 환경을 구축하여 기능 변경 시 즉각적인 반영이 가능하도록 하였다.
  
  
## Conclusion
 본 프로젝트는 학과 환경에 최적화된 인터렉티브 키오스크 시스템을 MSA 기반으로 설계·구현하였다. 각 기능을 독립적인 서비스로 분리함으로써 유지보수성과 확장성을 확보하였으며, 공지사항·공간 예약·프로젝트 전시·홍보 영상 재생 기능을 통합 플랫폼 형태로 제공한다. 또한 Raspberry Pi를 활용한 실물 키오스크 운영을 통해 실제 적용 가능성을 검증하였으며, GitHub Actions 기반 CI/CD 환경 구축으로 지속적인 배포 자동화를 실현하였다. 해당 시스템은 향후 타 학과 또는 기관에서도 활용 가능하도록 오픈소스 형태로 확장이 가능하다는 점에서 의의가 있다.
  
## Project Outcome
 본 프로젝트는 학과 내 키오스크 시스템 구축을 위한 기술적 기반을 마련하였으며, 공지사항·예약·영상·프로젝트 전시 기능이 포함된 통합 플랫폼 형태로 개발되었다. MSA 구조, PostgreSQL 기반 데이터 관리, Docker 및 GitHub Actions를 활용한 CI/CD 환경, Raspberry Pi 기반 실물 적용 가능성 등을 확보하였다.
